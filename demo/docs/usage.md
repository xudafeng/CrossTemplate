#Usage

('id','source')

#### Template

    wanpi.template(id,data,config);

#### Render

    wanpi.render(source,data,config);

#### Compile

##### compile to javascript

    wanpi.compile(id?,source,config);

##### compile to velocity

    wanpi.compileToVm(source,config);

##### compile to php

    wanpi.compileToPhp(source,config);

#### Config

    wanpi.config(config);
